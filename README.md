# 🐙 RevisonGrid
🐙 RevisonGrid is an personal planning system that turns goals into actionable daily execution.  Beyond priority management, habit tracking, and drag-and-drop scheduling, it features a smart auto-planning engine: for quantifiable goals, users simply choose working days per week, and the system automatically distributes tasks across the calendar.

> 告别繁琐的日历应用和涣散的备考计划。这是一款专为「多线程硬核备考」与「复杂项目推进」打造的极简前端追踪系统。
> 毕业季，考公、论文、找工作八手抓👋，仍然有条不紊地掌控你的每一项任务。


## ✨ 项目亮点 (Key Features)

* 📅 **一键智能排期**
  - 只需设定目标总量（如 20 套卷子），系统会自动在日历中生成 `项目 01` 到 `项目 20` 的序列。
  - 设置起始日和逢周几安排，一键生成自动排期，可随时在月历中检查计划安排是否合理。

* 🎯 **进度可视化**
  - 目标卡片内置 `+1`、`+5` 步进引擎与进度条，本月进度一目了然。
  - 支持自定义项目专属色彩，选定的颜色将自动渲染到下方追踪日历的对应文本中。
  
* 🖐️ **任务丝滑拖拽**
  - 计划赶不上变化？
  - 鼠标悬浮任务即可唤出拖拽手柄，支持任务自由拖拽。

* 🔒 **100% 纯本地隐私**
  - 支持一键将矩阵数据导出为CSV格式。
  - 无需注册，零服务器交互。
  - 所有数据实时保存在浏览器中，你的数据完全只属于你自己。

## 🎯 适用场景 (Use Cases)

本系统适合需要**长周期、高专注度、多项目并行**的场景：
1. **硬核升学与考公备考**：例如将行测 600 题、资料分析网课拆解到每日。
2. **雅思学习**：雅思听说读写四维度全方面排期与复盘，每日进度直观展示。
3. **日常自律与微习惯**：内置横向微习惯打卡矩阵（如健身有氧、专业软件操作练习），每日纯点击即可完成状态切换（✅ / ❌）。

1. **🌐 立即在线体验**：直接点击上方的 Live Demo 链接，在浏览器中即可开始规划你的当月目标。
